Cascading Style Sheets

Cascading Style Sheets ( CSS ) bahasa lembar gaya yang digunakan untuk menentukan presentasi dan gaya dokumen yang ditulis dalam bahasa markup seperti HTML atau XML (termasuk dialek XML seperti SVG , MathML atau XHTML ). [ 1 ] CSS adalah teknologi dasar World Wide Web , bersama dengan HTML dan JavaScript . [ 2 ] CSS dirancang untuk memungkinkan pemisahan konten dan presentasi , termasuk tata letak , warna , dan font . [ 3 ] Pemisahan ini dapat meningkatkan aksesibilitas konten , karena konten dapat ditulis tanpa memperhatikan presentasinya; memberikan lebih banyak fleksibilitas dan kontrol dalam spesifikasi karakteristik presentasi; memungkinkan beberapa halaman web untuk berbagi format dengan menentukan CSS yang relevan dalam file .css terpisah, yang mengurangi kompleksitas dan pengulangan dalam konten struktural; dan memungkinkan file .css untuk di-cache untuk meningkatkan kecepatan pemuatan halaman antara halaman-halaman yang berbagi file dan formatnya
